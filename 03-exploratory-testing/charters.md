# Charter 1

## Título
**Análisis del proceso de compra de productos**

## Misión
Analizar el recorrido completo de compra en JPetStore, desde la selección de productos hasta la confirmación final del pedido, utilizando tanto datos válidos como inválidos con el objetivo de identificar posibles fallos en:
- Validaciones
- Cálculo de precios
- Comportamiento del proceso de checkout

## Área principal explorada
- Carrito de compras  
- Proceso de checkout  
- Generación de órdenes  


# Charter 2

## Título
**Análisis del inicio de sesión y gestión de usuarios**

## Misión
Examinar el proceso de autenticación en JPetStore mediante el uso de:
- Credenciales válidas  
- Credenciales inválidas  
- Casos límite  

Con el fin de detectar posibles problemas en:
- Validaciones  
- Seguridad  
- Manejo de sesiones  

## Área principal explorada
- Login  
- Registro  
- Gestión de sesión
